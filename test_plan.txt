Тест-план: Калькулятор скидок
1. Ввод цены
- Ввела 1000 → отобразилось 1000.00
- Ввела 0 → ошибка «Цена должна быть больше 0!»
- Ввела -50 → ошибка, не даёт ввести
2. Ввод скидки
- Ввела 10 → отобразилось 10.00
- Ввела 100 → отобразилось 100.00
- Ввела 150 → ошибка, не даёт ввести
3. Расчёт
- Цена 1000, скидка 10 → итог 900, экономия 100
- Цена 2000, скидка 25 → итог 1500, экономия 500
- Цена 500, скидка 0 → итог 500, экономия 0
4. Каскадные скидки
- Добавила 10%, потом 20% при цене 1000 → итог 720, экономия 280
- Добавила 5%, потом 15% при цене 5000 → итог 4037.50, экономия 962.50
- Добавила только одну скидку 30% → итог 700, экономия 300 (работает как обычная)
5. Загрузка фото
- Нажала «Загрузить фото», выбрала picture.jpg → фото появилось
- Нажала «Удалить фото» → фото исчезло, написало «Нет фото»
- Попробовала загрузить не картинку → вылезла ошибка «Не удалось загрузить изображение»
6. Сохранение в историю
- Сделала расчёт, нажала «Сохранить» → вылезло сообщение, что сохранено
- Перешла на вкладку «История», нажала «Обновить» → запись появилась
- Сделала пустой расчёт, нажала «Сохранить» → ошибка «Сначала выполните расчёт!»
7. Очистка истории
- Нажала «Очистить историю» → спросило подтверждение
- Нажала «Да» → всё очистилось
- Нажала «Нет» → ничего не изменилось
8. Экспорт CSV
- Нажала «Экспорт CSV» → открылось окно сохранения
- Нажала «Экспорт CSV» при пустой истории → ошибка «История пуста!»
- Сохранила файл → он появился в папке
- Открыла файл → там были все записи из истории
9. Выход из программы
- Нажала крестик → спросило «Вы уверены?»
- Нажала «Да» → программа закрылась
- Нажала «Нет» → программа осталась открытой
